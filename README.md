# 📊 Data Analytics Portfolio – Entregables NUCLIO

Este repositorio recoge los 5 entregables desarrollados durante el programa en NUCLIO.  
Cada proyecto aborda diferentes competencias dentro del análisis de datos.

---

## 🐍 Entregable 1 | Python
Fundamentos de programación en Python aplicados a manipulación y tratamiento de datos.

## 📈 Entregable 2 | Statistical Thinking
Aplicación de conceptos estadísticos para la toma de decisiones basada en datos.

## 📊 Entregable 3 | Data Analysis
Análisis exploratorio y extracción de insights relevantes.

## 🗄 Entregable 4 | SQL para Data Analysts
Consultas y análisis sobre bases de datos relacionales.

## 📊 Entregable 5 | Dataviz
Visualización de datos orientada a comunicar insights de forma clara y efectiva.
